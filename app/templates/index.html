About the project

Many websites are currently performing user profiling through various techniques which recently have become very sophisticated. Actually, these techniques enable them to perform browser or even device fingerprinting.

The goal of this service is to collect similar data in order to better understand how they work and how they can be countered. Your contribution to the project is minimal in terms of time and resources, but invaluable. Just click on the button above to provide us the data of your browser.

How does it work?

All browsers have to conform to specific standards in order to render properly the webpages that you browse. However, there are many differences between them ranging from inherent; such as version and webkit, platform dependent; such as os version, to user dependent; such as fonts, languages etc. Collecting this data one can create a fingerprint for the browser to track and even profile the user.

The current webpage, when you decide to contribute and press the button, uses JavaScript and Flash (if installed) to collect such data from your browser and store it so that we can process it later.

How can I contribute?

Send us your data by just clicking on the button above. Use more browsers and other devices to enrich the sample. In every case we store one sample per browser and send you a cookie (browser one) so that we don't keep multiple copies of the same browser.

If you want, you can help in the backend as well! The code is open and you can extend it. Yes we know that we should unravel of the spaghetti code there, but ...

What do you mean process?

We may publish or share aggregated, statistical data from the collected data to facilitate privacy research in order to advance privacy-enhancing technologies. The scope of the project is not to track or profile users, but the contrary, to develop countermeasures.

When the project gets more mature, this site will collect data in order to create a pool of anonymised profiles which are going to be used by a tool (under development) to provide more anonymity. When this tool is developed, old data will be removed, so that the pool contains only profiles of the people who consend to share their data likewise. Clearly, at that point there will be proper alerts for the users. However, now we need the data to develop this tool.
